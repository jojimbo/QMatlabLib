csvreadCell

raw = csvreadCell(filename)

Reads mixed numeric and string data from a comma-separated-value file into a cell array.

Read data from text file into a cell array. File contains mixed text and numeric data in comma-separated-value format. File can have different numbers of entries on each row, those entries can be empty.

Inputs:

  • filename char

Outputs:

  • raw cell
